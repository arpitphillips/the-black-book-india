# Research Repository

This directory is designated for storing topical research.

## Instructions for AI Assistants

When the user asks you to perform research, you MUST adhere to the following organizational structure:

1. **Categorization**: Determine the broad category or topic of the research (e.g., `professions`, `technology`, `geography`).
2. **Folder Structure**: Create a subfolder for the category inside this `research` directory if it does not already exist (e.g., `research/professions/`).
3. **File Creation**: Create a Markdown file inside that category folder named after the specific subject of the research (e.g., `research/professions/photographers.md`).
4. **Content**: Populate the file with the requested research. Ensure the content is well-structured, comprehensive, and formatted in Markdown.

### Example Workflow
- **User Request:** "Research types of photographers."
- **Action:** 
  1. Identify the category: `professions`.
  2. Create folder: `research/professions/`
  3. Create file: `research/professions/photographers.md`
  4. Content: Add the research and different types of photographers into this newly created file.
